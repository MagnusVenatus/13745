Reporte Formal

Resumen de la Arquitectura

Se utilizó la plataforma de Azure para desplegar una máquina virtual con el sistema operativo Ubuntu. Esta máquina virtual fue configurada para alojar un servidor web utilizando Apache2, permitiendo así la publicación de una página web accesible desde Internet. La especificacion de la maquina virtual utilizada es la siguiente:

  • Nombre del equipo: Prueba1
  • Tipo de VM:Standar B1s
  • Sistema Operativo: Ubuntu Version 22.04
  • Arquitectura x64
  • Ram de 1GiB
  • 1 vCPU

Seguridad Implementada

Para garantizar el acceso seguro, se configuraron las reglas de red en Azure para abrir únicamente el puerto 80 (HTTP), permitiendo el tráfico web necesario y minimizando la exposición de otros servicios. Además, se utilizaron las credenciales seguras proporcionadas por Azure para el acceso administrativo a la máquina virtual.

Conclusión

El uso de Azure facilitó la creación y gestión de la infraestructura, permitiendo desplegar rápidamente un entorno seguro y funcional para alojar una página web. La correcta configuración de la red y el servidor Apache2 aseguró el acceso controlado y eficiente al servicio web.
